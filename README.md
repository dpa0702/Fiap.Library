# SISTEMA PARA BIBLIOTECA

# MÓDULO RESERVA DE LIVROS

## DESCRIÇÃO

---

Estes documento contem as informações do módulo de um sistema para biblioteca pública, para que os funcionários da recepção consigam gerenciar as locações dos livros.

 

## OBJETIVO

---

O objetivo do sistema é o desenvolvimento de um módulo que permite somente os funcionários gerenciem as reservas de livros seguindo as funcionalidades:

- Permitir que os funcionários cadastrem livros no sistema.
- Permitir que os funcionários editem dados dos livros.
- Permitir que os funcionários pesquisem livros disponíveis na biblioteca.
- Permitir que os funcionários cadastrem clientes no sistema.
- Permitir que os funcionários editem dados dos clientes.
- Permitir que os funcionários pesquisem clientes cadastros no sistema.
- Permitir que os funcionários cadastrem reservas de livros.
- Permitir que os funcionários visualizem o status de suas reservas.
